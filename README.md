## 🌟 About

Welcome to the source code of my personal portfolio website! This project is designed to showcase my journey as a Software Engineer and Flutter Developer. It features my professional experience, featured projects, and technical skills in a modern, interactive, and visually stunning layout.

The UI implements a dark theme with glassmorphism (frosted glass) effects and smooth scroll animations for a premium user experience.

## ✨ Features

- **Modern Aesthetic**: Dark theme, glassmorphism UI cards, and dynamic background glow.
- **Smooth Animations**: Integrated `framer-motion` for scroll-triggered reveals and micro-interactions.
- **Fully Responsive**: Optimized for seamless viewing on desktops, tablets, and smartphones.
- **Live Resume Integration**: Automatically reflects my latest internship experiences and technical skills.

## 🛠️ Tech Stack

This project was built using modern web development standards:

- **Framework**: [React.js](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/) (Lucide & FontAwesome)
- **Deployment**: GitHub Actions & GitHub Pages

## 🚀 Running Locally

Want to run this project on your own machine? Follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ajay0008-cloud/Ajay0008-cloud.github.io.git
   cd Ajay0008-cloud.github.io
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`.
